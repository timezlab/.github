# .github

Organization-wide default community health files and templates for **t0lab**.

Files in this repository are automatically applied to all repositories under the [t0lab](https://github.com/t0lab) organization that don't have their own versions.

## What's Included

| File | Purpose |
|------|---------|
| `ISSUE_TEMPLATE/bug_report.yml` | Bug report form |
| `ISSUE_TEMPLATE/feature_request.yml` | Feature request form |
| `ISSUE_TEMPLATE/question.yml` | Question form |
| `ISSUE_TEMPLATE/config.yml` | Issue template chooser config |
| `PULL_REQUEST_TEMPLATE/` | Default pull request template |
| `CONTRIBUTING.md` | Contribution guidelines |
| `CODE_OF_CONDUCT.md` | Community code of conduct |
| `SECURITY.md` | Vulnerability reporting policy |
| `FUNDING.yml` | GitHub Sponsors configuration |
| `profile/README.md` | Organization profile page |

## Overriding

Any repository can override these defaults by adding its own version of the file in its `.github/` directory.
